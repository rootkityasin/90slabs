# 90sX Portfolio - Environment Variables

## MongoDB Atlas Connection
# Get your connection string from: MongoDB Atlas > Database > Connect > Drivers > Node.js
# Format: mongodb+srv://<username>:<password>@cluster.xxxxx.mongodb.net/?retryWrites=true&w=majority
# 
# Add this to your .env.local file:
# MONGODB_URI=your_connection_string_here
